Drizzle JDBC
============
A BSD Licensed JDBC driver for MySQL and Drizzle.

Maven
-----
<pre>
`<dependency>
    <groupId>org.drizzle.jdbc</groupId>
    <artifactId>drizzle-jdbc</artifactId>
    <version>1.1</version>
</dependency>`
</pre>

Connection string
-----------------
`jdbc:drizzle://<user>@<host>:<port>/<database>`

or

`jdbc:mysql:thin://<user>@<host>:<port>/<database>`
